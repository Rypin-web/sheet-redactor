=== Sheet Redactor ===

Версия: 1.0.0
Автор: rypin-web
Лицензия: MIT

Описание:
Утилита для объединения XLSX таблиц, извлечения данных и составления сводных отчётов.

Возможности:
- 4 сценария обработки данных
- Поддержка старых и новых данных
- Автоматическое открытие результата в Excel
- Выравнивание столбцов по ширине

Требования:
- Windows 10/11
- Файлы .xlsx в папке с программой

Использование:
1. Положите файлы .xlsx в папку с программу
2. Запустите sheet-redactor.exe
3. Выберите сценарий и следуйте инструкциям
4. Результат появится в папке с программой

Примечание:
При первом запуске Windows может показать предупреждение безопасности.
Нажмите "Подробнее" → "Выполнить в любом случае".
